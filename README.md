# 📌 Project Overview

This project analyzes customer shopping behavior to uncover valuable business insights from transactional data. The workflow follows a complete analytics pipeline—from **data cleaning in Python**, **business analysis using MySQL**, to **interactive dashboard development in Power BI**.

---

# 🎯 Business Objective

The objective of this project is to understand customer purchasing behavior and answer business questions related to:

- Customer Segmentation
- Revenue Analysis
- Subscription Behavior
- Product Preferences
- Discount Impact
- Shipping Analysis
- Customer Demographics
